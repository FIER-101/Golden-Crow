# Golden Crow

一个使用 Godot 引擎和 GDScript 开发的游戏项目。

## 项目结构

```
Golden-Crow/
├── scenes/          # 游戏场景
├── scripts/         # GDScript 脚本
├── assets/          # 游戏资源（图片、音效等）
├── project.godot    # 项目配置
└── README.md        # 项目文档
```

## 快速开始

1. 用 Godot 4.0+ 打开项目
2. 在编辑器中运行主场景

## 开发指南

- 所有游戏脚本放在 `scripts/` 目录
- 场景文件放在 `scenes/` 目录
- 资源文件放在 `assets/` 目录
